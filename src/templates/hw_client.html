Assignments

On an assignment page, NewlineHW will generate a link for each submissin. This link will either trigger a clone, or be an external link (assuming the submitted homework is something like a codepen)

Submissions

On an submissin page, NewlineHW will generate a link for the submission, near the top of the page.

Upgrading

Run the following command to upgrade newline_hw

$ newline_hw upgrade
