This allows for easy cloning of PR's, Git Repos, Gists, and various other git related links from inside of Newline. For full documentation please read more about the underlying tool newline_hw here.

Installation
  1. macOS Sierra or greater
  2. Ruby 2.3.0 or greater
  3. Install newline_cli 0.2.5 or greater
  4. Log in to NewlineCLI
  5. Install newline_hw 1.1.4 or greater
    IMPORTANT Follow newline_hw install instructions!
